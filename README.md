# School Management System

A comprehensive School Management System with separate portals for Student/Parents and Admin/Staff.

## Quick Start (XAMPP)

1.  **Backend**: Ensure **Apache** and **MySQL** are running in XAMPP.
2.  **Database**: Create a database named `schol_management` in PhpMyAdmin.
3.  **Startup**: Double-click **`start_all.bat`** in the root folder to start the frontend portals.
4.  **Open**: Go to [http://localhost/school-management-system/](http://localhost/school-management-system/)

## Project Structure
- `backend/`: PHP API and authentication logic.
- `client-app/`: Student & Parent portal (Next.js).
- `admin-app/`: Admin & Staff dashboard (Next.js).

## Troubleshooting
- **Connection Refused**: Ensure `start_all.bat` is running and the terminal windows stay open.
- **Folder Listing**: The root `index.php` acts as the landing page. Clear your browser cache if you still see folders.
